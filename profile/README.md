# Underpass AI

Underpass AI builds open infrastructure for autonomous software engineering.

We focus on three concrete layers that can be used together or adopted separately:

| Repository | Purpose |
| --- | --- |
| [`swe-ai-fleet`](https://github.com/underpass-ai/swe-ai-fleet) | Multi-agent software engineering platform for planning, deliberation, and execution. |
| [`underpass-runtime`](https://github.com/underpass-ai/underpass-runtime) | Governed execution plane with isolated workspaces, tool invocation, artifacts, and policy controls. |
| [`rehydration-kernel`](https://github.com/underpass-ai/rehydration-kernel) | Deterministic context rehydration service and contracts for agentic systems. |

## What we build

- Multi-agent workflows for real software delivery
- Controlled execution infrastructure for tool-driven agents
- Deterministic context retrieval and rehydration
- Self-hostable services with Docker and Kubernetes deployment paths
